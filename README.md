
# 📊 SQL For Data Job Analysis — Portfolio Project  
## 🧩 Overview
This repository features my project from Luke Barousse’s SQL for Data Analysis course, where I explored real‑world job‑market data using advanced SQL techniques.
The project demonstrates how structured queries uncover salary and skill trends — transforming raw data into actionable insights for recruiters and analysts alike.

## 🎯 Objectives
Apply SQL fundamentals: joins, aggregations, window functions, and subqueries.

Analyze data‑analyst job postings to identify salary and skill patterns.

Translate query results into visual insights using Power BI and Python.

Build a recruiter‑friendly portfolio artifact highlighting analytical depth.

## 🧠 Key Insights
**1. SQL is universal** — featured in nearly every high‑paying data‑analyst role.

**2. Python dominates** — preferred for automation and data wrangling.

**3. Visualization tools matter** — Tableau and Power BI drive business storytelling.

**4. Cloud & collaboration tools** — Azure, AWS, GitLab, and Jira signal workflow maturity.

**5. Big Data & ML skills** — PySpark, Pandas, and DataRobot command top salaries.
